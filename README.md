# CutTime - Website Salon Tóc

## 📌 Mô tả
Đây là dự án frontend được xây dựng dựa trên giao diện mẫu có sẵn.  
Website mô phỏng trang landing page cho salon tóc, bao gồm các phần như dịch vụ, kiểu tóc và đánh giá khách hàng.

## 🚀 Công nghệ sử dụng
- Vue 3
- Vite
- HTML, CSS

## 🔗 Demo
https://cuttime-8gzv.onrender.com/

## 📂 Tính năng
- Giao diện responsive (hiển thị tốt trên nhiều thiết bị)
- UI rõ ràng, dễ nhìn theo mẫu thiết kế
- Các section: Banner, Dịch vụ, Kiểu tóc, Đánh giá khách hàng
